# 英単語帳 | Vocabulary Cards

スマホで学ぶ英単語帳 - 6000語以上の英単語をフラッシュカード形式で効率的に学習

## 🎯 機能

- **フラッシュカード学習**: タップでカードを裏返し、単語と意味を確認
- **進捗管理**: 「覚えた」「覚えていない」で学習状況を記録
- **単語一覧**: 検索・フィルター機能付きの単語リスト
- **学習統計**: 達成率をグラフで可視化
- **スワイプ操作**: 左右スワイプでカード切り替え
- **シャッフル**: ランダムな順序で学習
- **ダークモード対応**: システム設定に自動追従
- **オフライン対応**: ローカルストレージで進捗保存

## 📱 スマホ最適化

- モバイルファーストのレスポンシブデザイン
- タッチ操作に最適化されたUI
- セーフエリア対応（ノッチ等）
- PWA対応（ホーム画面に追加可能）

## 🚀 使い方

1. [GitHub Pages](https://halc8312.github.io/CopEng/)にアクセス
2. カードをタップして裏面（意味）を確認
3. 「覚えた」または「覚えていない」をタップ
4. 左右スワイプまたはボタンで次のカードへ

## ⌨️ キーボードショートカット

| キー | 動作 |
|------|------|
| ← | 前のカード |
| → | 次のカード |
| Space | カードを裏返す |
